# 介绍<br>
## 项目名称： <br>
  尚硅谷ego百战商城。 项目介绍： 综合性的 B2C 电子商务平台。用户可以在系统中通过搜索商品、查看商品详情、加入购物车、购买商品并生成订单完成购物操作。<br>
<br>
## 软件架构：<br>
### 百战商城共分为两部分：<br>
  商城后台管理系统： 主要实现对商品、商品分类、规格参数、CMS 等业务的处理。<br>
  商城前台系统： 主要提供用户通过访问首页，完成购物流程的处理。<br>
